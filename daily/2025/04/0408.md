**つまづいたところ・学び：**
- 作業中に割り込み作業する時に便利なstash
```
// stashする
$ git stash

// 追跡されていないファイルがある場合は一度コミットするか以下コマンド
$ git stash --include-untracked "WIP: xxx"
// 省略版
$ git stash -u

// 確認
$ git stash list

// 〜別作業〜
// 作業へ戻る
$ git stash apply stash@{0}

// 作業をもとに戻すと同時に消すことも可能
$ git stash pop stash@{0}

// stashしたものを削除
$ git stash drop stash@{0}
```
